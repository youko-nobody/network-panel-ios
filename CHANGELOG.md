# Changelog

## 0.1.24 - 2026-06-15

- 修复 iPad 横屏首页左右卡片底部未对齐的问题。
- 重新设计 iPad 横屏地区延迟横条，让节点与延迟信息更规整。

## 0.1.23 - 2026-06-15

- 修复 0.1.22 的 Release 构建问题，保留 iPad 横屏双栏工作台布局。

## 0.1.22 - 2026-06-15

- 优化 iPad 横屏首页布局，采用居中双栏工作台。
- 实时网速与控制台按 55 / 45 比例展示，地区延迟改为底部横向卡片。

## 0.1.21 - 2026-06-15

- 修复 0.1.20 的构建兼容问题，确保 Release 能正常生成 IPA。

## 0.1.20 - 2026-06-15

- 首页新增本次流量上限入口，可直接设置 MB / GB / TB。
- 首页速率上限改为可点击手动输入，逻辑与 Android 端保持一致。
- 线路管理支持编辑已有线路。
- iPad 竖屏改为 iPhone 比例的窄布局居中显示。

## 0.1.19 - 2026-06-14

- 软件处于前台时保持屏幕常亮，切到后台后恢复系统自动锁屏。
- Release 页面改为展示本次更新内容，不再重复功能列表。

## 0.1.18 - 2026-06-14

- 将最低系统版本降到 iOS / iPadOS 16.0，支持 iPadOS 16.3.1 安装。

## 0.1.17 - 2026-06-14

- 移除首页内置项目地址入口，让应用界面更专注。
